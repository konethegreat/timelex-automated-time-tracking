# Contributing to TimeLex

Thank you for your interest in the TimeLex Automated Time Tracking system. 

## Code of Conduct
By participating in this project, you agree to abide by our Code of Conduct.

## How Can I Contribute?
As this project is currently part of a technical assessment and a private portfolio piece:
1. **Reporting Bugs:** Please open an issue on GitHub with a clear description and steps to reproduce.
2. **Feature Requests:** Use the Issues tab to suggest new ideas.
3. **Pull Requests:** PRs are currently **not** being accepted for the main branch to ensure the integrity of the assessment submission. 

## Development Standards
If you are reviewing the code:
- The project follows a modular architecture for scalability.
- All database interactions are handled via Prisma to ensure type safety.
- Security and data privacy are prioritized, especially regarding municipal and legal data.
